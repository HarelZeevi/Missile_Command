# Missile_Command
I've recreated the familiar retro game "Missile Command" On Assembly. Enjoy!

Requirments:
  Dosbox (free 8086 emulator)
 
 How to start playing:
  1. download the files into TASM/BIN directory
  2. run dosbox 
  3. write in the prompt: "game.exe"
 
 Enjoy!
